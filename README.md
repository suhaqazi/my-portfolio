## About

My personal portfolio website showcasing my projects, skills, and experience as a Computer Science student at Trent University.

**Features:**
- 🎨 Custom dark theme with purple/pink color palette
- 📱 Fully responsive design
- ⚡ Interactive project timeline with modal popups
- 📧 Working contact form with Web3Forms API
- 📊 Vercel Analytics for visitor tracking
- 🖱️ Smooth animations with Framer Motion
